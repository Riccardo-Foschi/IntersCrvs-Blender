Download the zip file and drag and drop it in Blender viewport to install.
this adds a new icon on the menu bar of the 3D viewport and a new entry in the right click menu.

- Blender plugin that automatically creates vertices in curves (and meshes that contain only edges). 
- This helps for snapping at intersections.

- it also remove vertices in colinear lines and removes unnecessary vertices in circles.
- This helps when the user wants to clean up the vertices generated with previous command.
